# MWU Campus Quest

Welcome to the Madda Walabu University Game Project!

## Getting Started

### 1. Prerequisites
Make sure you have **Node.js** installed on your computer.

### 2. Installation
Open your terminal in this folder and run:
```bash
npm install
```

### 3. Running the App
To start the development server:
```bash
npx expo start
```
- Scan the QR code with the **Expo Go** app on your Android (Play Store) or iOS (Camera app) device.

## Project Structure
- **App.js**: The main entry point where navigation is set up.
- **src/screens/**: Contains the screens of your app.
  - `HomeScreen.js`: The welcome screen.
  - `GameScreen.js`: The quiz logic and display.
  - `ResultScreen.js`: Shows the final score.

## How to Customize
1. **Add Images**: Put photos of MWU buildings in an `assets` folder.
2. **Update Questions**: Open `src/screens/GameScreen.js` and edit the `QUESTIONS` array to include real data and link to your images.
